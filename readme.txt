Image Gallery Management Plugin for Photographers

== Description ==

*für deutsch: weiter unten lesen*

Miquado Gallery is a powerful image gallery management plugin for photographers. 

The plugin features fast and easy upload of images by drag&drop from the desktop, automatic setting of title, keywords and description from the image meta data, watermark, sorting galleries, copying and moving images. Miquado Gallery makes it easy to organize galleries and to re-arrange them at any time.

Professional features can be added to Miquado Gallery, such as:

* background music for slideshows
* password protection of galeries
* allow customers to download selected images from published galleries
* allow customers to create image collections and send you 
* selling images as downloads or prints directly from your blog (with standard theme).
* paypal online payment

For more information visit [Miquado](http://www.miquado.com)

*DEUTSCH*

Miquado Gallery ist eine leistungsfähiges Bildergalerie-Plugin für Fotografen.

Das Plugin ermöglicht den schnellen und einfachen Upload von Bildern per Drag&Drop vom Desktop, die automatische Verschlagwortung aus den Bild-Metadaten, integriert ein Wasserzeichen und bietet Sortierfunktionen sowie die Möglichkeit, Bilder zwischen Galerien zu kopieren und zu verschieben. 

Weitere professionelle Features, die zu Miquado Gallery hinzugefügt werden können:

* Hintergrund-Musik für emotionale Slideshow
* Passwort-Schutz für einzelne Galerien
* Freigabe von Galerien für den direkten Download von Bildern
* Kunden ermöglichen, Bilder-Kollektionen zusammen zu stellen 
* Bilder als Download oder Print verkaufen (mit einem Standard-Theme).
* Paypal-Zahlungssystem

Weitere Informationen finden Sie auf [Miquado](http://www.miquado.com)


== Installation ==

1. Go to [Plugins] -> [Add New] -> [Upload]
2. Select the downloaded ZIP file for uploading, installing and activation

= How to add Images =

1. Click [Overview] (the default page)
2. Click [+] to create a category
3. Click [+] to create a gallery in that category
4. Click [0 Images] to get to the images upload page

= How to publish a gallery or categories =

* Create a post or a static page
* Set the cursor in the text where you want the gallery
* Click the Miquado Gallery editor button, this will create a spaceholder link
* Click to the spaceholder link
* Select [Gallery] to display a single gallery or [Categories] to show one or more categories
* Publish your post or page

= Using the Widget = 

The Miqudo Gallery Widget has multiple functions

1. Display a category/gallery Navigation
2. Show thumbs 
3. Show the shopping cart link

* Navigation Widget *

Two parameters can be used on the navigation widget:

targetid: lets the navigation know to which static page it shall link.
categories: defines the category numbers that are shown in the navigation (default is "all")
Example: 
`targetid=14 categories=3,6,10

* Thumbs widget *

If set accoringly in the Miquado Gallery configuration, thumbnail images of a gallery are shown in the widget. No parameters required.

For more information please refer to the handbook available from Miquado.
